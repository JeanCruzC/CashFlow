El **flujo de caja neto mensual** responde una pregunta crítica:  
si hoy cerraras el mes, ¿te quedó dinero libre o terminaste financiando tus gastos con deuda?

## Fórmula base

`Flujo de caja neto = Ingresos del mes - Gastos del mes`

No necesitas un modelo complejo para empezar. Necesitas consistencia:

1. Registrar todos los ingresos que realmente ingresaron a tus cuentas.
2. Registrar todos los gastos, incluidos pagos automáticos.
3. Clasificar cada movimiento para analizar por categoría.

## Estructura mínima para operar

Con una estructura simple puedes tener visibilidad en menos de una semana:

| Campo | Ejemplo |
| --- | --- |
| Fecha | 2026-02-20 |
| Cuenta | Banco principal |
| Categoría | Alimentación |
| Monto | -145.80 |
| Descripción | Compras semanales |

## Errores comunes que distorsionan el resultado

- Mezclar transferencias entre cuentas como si fueran gastos.
- No incluir pagos de tarjeta ya consumidos en el mes.
- Cargar movimientos sin categoría.
- Revisar el flujo solo al final del mes.

## Qué decisión te habilita esta métrica

Cuando mides el flujo de caja semanal y mensual puedes:

1. Corregir gastos discrecionales antes de cerrar el período.
2. Ajustar presupuesto por categoría con base real.
3. Definir una meta de ahorro sostenible.

Un flujo de caja positivo y estable es la base para luego construir métricas más avanzadas como tasa de ahorro o patrimonio neto.
